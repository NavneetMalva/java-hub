Summary of Autowiring Annotations in Spring Boot
Autowiring Type	Description	Annotation(s)
By Type	- Default type-based injection	@Autowired
By Name - Specifies a specific bean by name	@Qualifier
Primary - Sets a preferred bean for injection	@Primary
Constructor Injection - Injects via a constructor (recommended)	@Autowired
Lazy Initialization - Injects dependency only when needed	@Lazy
